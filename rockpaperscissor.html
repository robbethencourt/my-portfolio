using Firebase to create a multiplayer rock paper scissor game with a UI optimized for mobile

html / css / javaScript / firebase
Created 2016

Real-Time Trash Talk

Real-time updates through Firebase was the focus of this project. The score needed to be updated, but also the ability to talk trash to your opponent needed to update realtime. A feature my wife and I tested extensively through the development process.

Mobile RPS

Playing this on a phone seemed like the best way to enjoy this game. It's got the ability to play in quick spurts and quickly responds with updates through Firebase, both of which are ideal for mobile gaming.
